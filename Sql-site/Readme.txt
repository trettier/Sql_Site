main.py - для запуска сайта
После создания турнира можно начать или продолжить его решать
Чтобы ответить на задание нужно нажать кнопку "Дать ответ", написать его, нажать
"Проверить", обновить страницу, и если он верный, появится надпись "Решено"  

Информация о пользователях и заданиях хранится в blogs.db, о турнирах и баллах в list_ctf.json
Аккаунт, с которого можно создавать турнир:
nicrname = admin
email = admin@mail.ru
parol = 123
